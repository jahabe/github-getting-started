## GitHub 
A platform built on Git that lets everyone: 
- track changes to each other's code
- collaborate with others
- maintain a history of our projects

## Core concepts of the GitHub flow: 
- Repositories store my project files
- Branches let me experiment safely
- Commits + pushes save work to the remote repo 

## Pull requests
How we propose and review changes before merging them into the main branch

## Issues 
help track tasks and bugs
